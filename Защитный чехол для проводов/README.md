# Защитный чехол-фиксатор зарядного кабеля USB-C

Жёсткий протектор-фиксатор (strain relief) для зарядного кабеля **USB-C**
(iPhone 16 / MacBook Air). Бубышка (овермолд штекера) заводится в гнездо с
торца и чуть торчит наружу, провод укладывается в тоннель сверху и держится
зубцами. Печать из **PLA**.

**Версия 5** — переработана по результатам печати: **плоская подошва** (печать
**без поддержек**), тоннель Ø4.1, фиксация зубцами, посадка бубышки — вариант №3.

| | V1 — чехол | V2 — серпантин |
|---|---|---|
| Низ | плоская подошва | плоская подошва (сегменты-пружина) |
| Верх над пружиной | сплошной | сквозные разрезы (гибкая пружина) |
| Фиксация провода | зубцы у паза | зубцы + серпантин |
| Гнездо | крыша + прорезь 4.1 | крыша + прорезь 4.1, удлинённое |
| Габарит, мм | 52.0 × 14.2 × 10.2 | 56.0 × 14.2 × 9.4 (тоньше) |
| Чертёж | ![V1](<Готовые модели/V1 — чехол/CableProtector_drawing.png>) | ![V2](<Готовые модели/V2 — пружина/CableProtector_Spring_drawing.png>) |

> **Размеры разъёма (верифицировано).** Штекер USB-C нормирован USB-IF:
> 8.34 × 2.56 мм, глубина вставки ≈ 6.65 мм. Овермолд («бубышка») стандартом
> **не** нормируется — взяты замеры пользователя: **10 × 6 × 17 мм**.

## Общие решения v5 (обе версии)

* **Плоская подошва (D-сечение):** низ плоский, верх скруглён. Деталь стоит на
  столе всей плоскостью → **печать без поддержек** (раньше круглый низ был
  подвесным и требовал поддержек, которые отрывались и портили поверхность).
* **Тоннель Ø4.1** (уменьшен с 4.4) — одинаковый в обеих версиях, прямой, без
  перегородок.
* **Зубцы-фиксаторы** — скруглённые бугорки у кромок верхнего паза (без острых
  углов, не выступают за корпус): провод проталкивается мимо них в тоннель и
  слегка прижат сверху; внутри тоннеля гладко. Между зубцами паз открыт.
* **Гнездо бубышки — вариант №3** (зазор **0.12/ст**, R2.5): лучший по тесту
  образцов. Сверху **крыша со сплошной прорезью 4.1 мм** по всей длине (без
  уступов и перегородок): бубышка заводится с **торца** под крышу (крыша держит
  её от выскока), провод проталкивается в прорезь сверху.

## V2 — серпантинный чехол

`Готовые модели/V2 — пружина/` — над тоннелем **сквозные поперечные разрезы
поочерёдно слева/справа**: получается гибкая цепочка сегментов на чередующихся
перемычках (полноценная пружина, гасит перегиб). Низ при этом — отдельные
**плоские пятки** каждого сегмента (печать без поддержек, сплошного низа нет).
**Гнездо удлинено** (SOCK_END 14→20, общая длина 56) — провод и бубышка заходят
до упора.

## Тестовые образцы гнезда бубышки

прорезью и **плоской подошвой**. По тесту выбран **№3** (он зашит в обе модели).

| № | зазор/ст | R, мм | гнездо Ш×Т, мм | результат теста |
|---|---|---|---|---|
| 1 | +0.25 | 1.5 | 10.50 × 6.50 | микролюфт, выпадает |
| 2 | +0.18 | 2.0 | 10.36 × 6.36 | почти без люфта, но не держит |
| 3 | +0.12 | 2.5 | 10.24 × 6.24 | **лучший — выбран** |
| 4 | +0.06 | 2.8 | 10.12 × 6.12 | не залезает |
| 5 |  0.00 | 3.0 | 10.00 × 6.00 | не залезает |

## Печать (PLA)

* **Плоской подошвой на стол, прорезью вверх.**
* Крыша над гнездом печатается коротким мостиком (полки ~3 мм) — на FDM проходит
  без поддержек; при желании включить поддержку только под крышей.
* Для V2 (сегменты-пятки) полезен **brim 4–5 мм** для сцепления со столом.
* Сопло 0.4, слой 0.2, стенок 3, заполнение 30–40 %.

## Скрипты (FreeCAD 1.1 Python API)

```bash
FC="C:/Program Files/FreeCAD 1.1/bin"
"$FC/freecadcmd.exe" Скрипты/build_cable_protector.py    # V1
"$FC/python.exe"     Скрипты/make_drawing.py
"$FC/freecadcmd.exe" Скрипты/build_boot_test_coupons.py  # 5 образцов
"$FC/freecadcmd.exe" Скрипты/build_spring_protector.py   # V2
"$FC/python.exe"     Скрипты/make_drawing_spring.py
```

Ключевые параметры (вверху скриптов): `BORE` (тоннель), `TOOTH_*` (зубцы),
`GAP_BOOT`/`BOOT_R` (посадка бубышки), `SOCK_END`/`TOTAL_L` (длины). V2:
`PITCH`/`BRIDGE_W`/`SPINE` (рельеф/подошва).
