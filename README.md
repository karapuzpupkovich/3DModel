# Репозиторий 3D-моделей (3DModel)

В данном репозитории собраны проекты 3D-моделей, разработанные для изготовления на 3D-принтере (FDM-печать, пластики PETG/ABS/ASA).

## Содержимое репозитория

Репозиторий состоит из двух основных проектов:

1. **[Панель для Nissan tiida](file:///c:/3DModel/Панель%20для%20Nissan%20tiida)** — Накладка (переходная панель) на центральную консоль автомобиля Nissan Tiida.
2. **[Шкафчик Никиты (Сказка м.г.)](file:///c:/3DModel/Шкафчик%20Никиты%20(Сказка%20м.г.))** — Модульная сотовая система для хранения детской обуви в шкафчик детского сада.

---

### 1. Накладка на панель Nissan Tiida
* **Путь к проекту:** [Панель для Nissan tiida](file:///c:/3DModel/Панель%20для%20Nissan%20tiida)
* **Назначение:** Закрытие проема в центральной консоли между экраном мультимедиа и блоком климат-контроля.
* **Конструктивные особенности:**
  - **Форма:** Трапециевидная (ширина верхнего края `293.0 мм`, нижнего `287.0 мм`, высота по центру `26.0 мм`).
  - **Толщина стенки:** `2.5 мм` (для жесткости).
  - **Крепление:** Фиксация за счет двух усиленных боковых защелок-клипс (ширина `10.0 мм`, длина вылета `8.0 мм`), развернутых на 90 градусов, с зубом наружу. Они защелкиваются за внутренние грани обшивки торпеды.
  - **Углы:** Наружные углы панели скруглены радиусом `1.0 мм` для плавного стыка с консолью.
* **Рекомендации по изготовлению:** Термостойкий пластик (PETG, ABS, ASA) во избежание деформации на солнце. Печать лицевой стороной вниз с поддержками для защелок.

---

### 2. Модульный обувной органайзер (Соты)
* **Путь к проекту:** [Шкафчик Никиты (Сказка м.г.)](file:///c:/3DModel/Шкафчик%20Никиты%20(Сказка%20м.г.))
* **Назначение:** Сборный органайзер из **5 сот** для детского шкафчика в раздевалке детского сада (размер обувной секции: $290 \times 200 \times 334$ мм).
* **Конструктивные особенности:**
  - **Ячейка-сота:** Внешний габарит по ширине `125.0 мм`, высота `96.0 мм`, длина `200.0 мм`. Толщина стенок `2.3 мм`.
  - **Скругления торцов (3D):** Безопасные скругления по всему периметру краев радиусом `0.8 мм` (чтобы ребенок не поцарапался).
  - **Соединение «паз-гребень» (Dovetail):** Замки на наклонных гранях. Для печати пластиком **PETG** зазоры уменьшены до **0.15 мм на сторону** (ширина паза `25.3 / 31.7 мм` при ширине гребня `24.0 / 31.0 мм`), обеспечивая плотную посадку с натягом и предотвращая легкое соскальзывание.
  - **Перфорация:** Безопасные скругленные шестигранные отверстия радиусом `2.5 мм` с шагом `7.0 мм` для вентиляции обуви.
  - **Полная сборка шкафа (`Cabinet.FCStd`):** Включает параметрическую дверцу (угол открытия задается в таблице `Spreadsheet` переменной `DoorAngle`) и 5 сот, стоящих на полке с гарантированным верхним зазором `8.0 мм` (высота сборки `192.0 мм` при высоте проема `200.0 мм`).
* **Скрипты автоматизации (FreeCAD Python API):**
  - `build_single_cell.py` — генерация твердотельной 3D-модели соты.
  - `build_cabinet.py` — сборка полного шкафа с сотами.
  - `check_honeycomb_overlap.py` — автоматическая проверка наложений соседних сот в сборке.
  - `render_dimensioned_views.py` — генерация чертежа ячейки в ортогональных и изометрической проекциях с размерами.
