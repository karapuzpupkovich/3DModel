# Профиль печати для HoneycombCell_large.stl (Bambu Lab A1, сопло 0.4)

Файл: `0.28 Соты-large прочно @BBL A1.json` — профиль **процесса** (Process) для
Bambu Studio. Деталь почти целиком стенки 2.3 мм + перфорация, поэтому профиль
заточен: высота слоя ради скорости, стенки Arachne ради прочности, без верхней
крышки и без инфилла.

## Импорт в Bambu Studio
1. `Файл → Импорт → Импортировать пресеты…` → выбрать
   `0.28 Соты-large прочно @BBL A1.json`.
2. Слева в выпадающем списке **Process** выбрать `0.28 Соты-large прочно`.
3. Принтер — Bambu Lab A1 (0.4), стол по своему.

Если импорт не подхватит родителя (другая версия Studio) — просто выставь
значения вручную, они ниже.

## Что в профиле (и зачем)
| Параметр | Значение | Зачем |
|---|---|---|
| Высота слоя | 0.28 (можно 0.30 в Orca) | главный выигрыш по времени |
| Генератор стенок | **Arachne** | стенка 2.3 не кратна 0.42 — ровно заполнит без щелей |
| Стенки (петли) | 5, ширина 0.45 | заполняют всю 2.3 мм = вся прочность |
| Заполнение | 0 % | стенки смыкаются, инфилл не нужен |
| Верх / низ | 0 / 4 слоя | корзина открытая — без сплошного верха |
| Поддержки | выкл | перфорация = вертикальные стенки |
| Brim | 5 мм | деталь 200 мм высотой, чтоб не опрокинулась |
| Дуги (arc fitting) | вкл | чисто и быстро по круглым отверстиям |
| Скорость стенок | внеш. 150 / внутр. 250 | чистый узор перфорации |

## Температуру задать в FILAMENT (её нет в профиле процесса)
Вкладка **Filament → Температура**, поднять сопло на **+5–10 °C** для прочного
спекания по Z (деталь грузится на изгиб):
- **PETG** (рекомендую для обувной корзины): сопло **245 °C**, стол 70–80, обдув 40–50 %.
- **PLA**: сопло **215 °C**, стол 55–60, обдув 100 %.
