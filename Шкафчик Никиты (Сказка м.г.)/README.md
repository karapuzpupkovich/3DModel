Этот документ содержит все наблюдения, расчеты размеров детского шкафчика в детском саду, технические параметры и фундамент для проектирования модульных органайзеров.

## 2. Расчет габаритных размеров шкафчика

На основе анализа приняты следующие габариты:
*   **Внешние габариты корпуса:** Высота **1295 мм** (без ножек), Ширина **322 мм**, Глубина **350 мм**.
*   **Внутренняя ширина:** **290 мм** (322 мм внешняя - 32 мм боковые стенки ЛДСП 16 мм).
*   **Внутренняя глубина:** **334 мм** (с учетом задней стенки 16 мм).
*   **Внутренняя высота обувной секции:** **200 мм** (от Z=16 до Z=216 мм).

## 3. Инструкция по управлению дверью (Механизм открывания)

Для удобства визуализации и проверки собираемости мы сделали дверцу шкафчика параметрической. Она вращается вокруг левой оси петель.

### Как открыть или закрыть дверь в FreeCAD:
1.  Откройте файл модели `Cabinet.FCStd` в FreeCAD.
2.  В дереве проекта слева найдите объект **`Spreadsheet`** (он находится в самом верху списка сборки).
3.  Дважды кликните на него, чтобы открыть таблицу параметров.
4.  В ячейке **`DoorAngle`** (столбец **B**, строка **1**) введите желаемое значение угла в градусах:
    *   `0` — дверь полностью закрыта.
    *   `45` — дверь приоткрыта наполовину.
    *   `90` — дверь открыта под прямым углом.
    *   `110` — максимальный угол открытия.
5.  Нажмите `Enter` и вернитесь на вкладку 3D-вида `NikitasCabinet` — модель автоматически перестроится с новым углом открытия двери.

## 4. Модульный сотовый органайзер для обуви (Фаза 3 — 5-блочный концепт с прямым опиранием)

В нижнюю обувную секцию ($290 \times 200 \times 334$ мм) спроектирована и встроена модернизированная модульная система из **5 ячеек-сот** в staggered (шахматном) порядке (2 по краям и 1 по центру), которые опираются непосредственно на нижнюю полку шкафчика (без использования подложки).

### Геометрические параметры и расчет стыковки:
*   **Максимальная ширина соты по углам ($2 \cdot a_{out}$):** **125.0 мм** ($a_{out} = 62.5$ мм).
*   **Смещение по горизонтали ($\Delta X$):** **77.5 мм** ($a_{out} + b_{out} = 62.5 + 15.0 = 77.5$ мм для идеальной стыковки).
*   **Ширина плоской горизонтальной грани ($2 \cdot b_{out}$):** **30.0 мм** ($b_{out} = 15.0$ мм).
*   **Высота соты ($2 \cdot r_{out}$):** **96.0 мм** ($r_{out} = 48.0$ мм).
*   **Радиус скругления внешних углов ($R_{c\_out}$):** **6.0 мм**.
*   **Толщина стенок соты:** **2.3 мм** — абсолютно равномерная по всему профилю, сгенерированная методом CAD 2D-смещения (`makeOffset2D(-2.3)`).
*   **Длина ячейки:** **200 мм**.
*   **Габариты проема:** Ширина **118.5 мм** (по внутренним углам), Высота **91.4 мм** (по внутренним плоским граням) — максимально просторный проем под детские кроссовки и ботинки.

### Соединение «Шип-Паз» (Трапециевидное ласточкино гнездо / Dovetail Joint):
*   Соединения выполнены в виде прочных трапециевидных замков типа «ласточкин хвост», расположенных строго по центру наклонных slanted-граней ячеек.
*   **Шип (Male):** Выступ высотой 3.0 мм, ширина у основания 24.0 мм (со скосом к v=-0.5 мм внутри стенки до 23.0 мм), ширина у вершины 30.0 мм. Вершинные/наружные углы закруглены по радиусу **0.4 мм**, базовые/внутренние углы оставлены острыми для монолитного слияния со стенкой.
    *   Расположен на правых slanted-гранях **0 и 5** (нормальные углы $\theta_0$ и $360^\circ-\theta_0$).
*   **Паз (Female):** Вырез глубиной 3.2 мм, ширина у входа (основания) 24.3 мм, ширина у дна (вершины) 30.7 мм. Входные/наружные углы скруглены по радиусу **0.4 мм** для легкого захода шипа и детской безопасности, дно паза (внутренние углы) оставлено острым.
    *   Расположен на левых slanted-гранях **2 и 3** (нормальные углы $180^\circ-\theta_0$ и $180^\circ+\theta_0$).
    *   Обеспечивает **зазор 0.3 мм по ширине** для легкой сборки скольжением.
*   **Усиление стенки за пазом:** Напротив пазов с внутренней стороны соты выполнена утолщающая площадка шириной 48.0 мм (у основания) и глубиной до 4.8 мм (остаточная стенка за пазом составляет **1.6 мм**). Все 4 угла площадки закруглены по радиусу **2.0 мм** для создания плавного органического профиля без концентраторов напряжений.
*   **Безопасные торцы (Скругление 3D):** Все 48 наружных и внутренних торцевых ребер ячейки (на переднем торце Z=0 и заднем торце Z=200 мм) скруглены по радиусу **0.8 мм**, чтобы ребенок не мог поцарапаться при укладывании обуви.
*   **Горизонтальные грани (Face 1 - верхняя, Face 4 - нижняя):** Полностью плоские (без шипов/пазов) для идеального прилегания к полке и в вертикальных стыках ячеек.

### Сотовая перфорация:
*   Скругленные hex-отверстия радиусом **2.5 мм** (уголки гексагонов скруглены по радиусу **0.8 мм** в 2D) для предотвращения острых граней.
*   **Сетка отверстий:** 27 рядов по длине Z с шагом 7.0 мм (от Z=9.0 до 191.0 мм), отступ от краев торцов составляет $\ge 9.0$ мм.
*   **Размещение по граням (строгий отступ $\ge 5.0$ мм):**
    *   **Плоские грани (1 и 4):** Шахматный паттерн из 2 колонок на четных рядах ($u = \pm 3.5$ мм) и 3 колонок на нечетных рядах ($u = 0.0, \pm 7.0$ мм). Минимальный отступ от краев $\ge 5.5$ мм.
    *   **Наклонные slanted грани (0, 2, 3, 5):** По 1 колонке с каждой стороны от замка в шахматном порядке: $u = \pm 23.5$ мм на четных рядах и $u = \pm 25.0$ мм на нечетных рядах. Минимальный зазор до замков $\ge 5.65$ мм, до угловых скруглений $\ge 6.25$ мм, что исключает любые трещины.

### Ориентация и монтаж (Исключение коллизий):
*   Все 5 ячеек ориентированы в одинаковом стандартном положении, собираются без необходимости поворота отдельных ячеек.
*   **Прямое опирание на полку:** Нижняя грань ячеек плоская, соты жестко стоят на полке шкафа без выступов.
*   **Верхний зазор:** Высота 5-блочной сборки составляет **192.0 мм** ($4 \cdot r_{out}$). С учетом нижней точки $Z = 16.0$ мм верх сот находится на уровне $Z = 208.0$ мм. Верхняя полка шкафа находится на уровне $Z = 216.0$ мм.
*   Благодаря этому остается **8.0 мм чистого гарантированного зазора** сверху, исключая любые коллизии с верхней полкой при установке.
*   **Боковые зазоры:** Ровно **5.0 мм** с левой и правой сторон (размещение от $X = 21.0$ до $X = 301.0$ мм в 290 мм пространстве от $X=16.0$ до $X=306.0$ мм).

## 5. Рекомендации по 3D-печати

1.  **Ориентация соты:**
    *   Располагать вертикально (на торце). Поддержки не требуются.
    *   Использовать внешнюю кайму (**Brim**) 8-10 мм для надежного сцепления со столом.

2.  **Материал:**
    *   **PETG** — идеален по гибкости и прочности замков.

3.  **Настройки слайсера:**
    *   Слой 0.20 – 0.28 мм, 3 периметра (стенки сопла 0.4 мм) для монолитности.
